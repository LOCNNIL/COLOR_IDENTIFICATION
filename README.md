# COLOR_IDENTIFICATION
In this repository, using Open CV python module, I make a simple color identification algorithm using the Webcam


## Prerequisites
Necessary Libraries:
* Open CV
* Numpy

## Python version
* [Python 3.7.x](https://www.python.org/downloads/release/python-379/)
